# C# Envanter Takip Sistemi

Bu proje, C# dilinde geliştirilmiş bir envanter takip sistemi uygulamasıdır. MSSQL veritabanı kullanılarak envanter öğelerinin yönetilmesini sağlar.

## Başlangıç

Projenin yerel bir makinede çalıştırılması için Visual Studio veya benzeri bir IDE kullanılabilir. Projeyi açtıktan sonra, MSSQL veritabanını kurarak ve gerekli bağımlılıkları yükleyerek projeyi çalışır hale getirebilirsiniz.

## Gereksinimler

- .NET Framework
- Microsoft SQL Server
- Visual Studio IDE veya benzeri bir geliştirme ortamı

## Kurulum

1. Bu depoyu yerel makinenize klonlayın.
2. MSSQL veritabanını oluşturun ve projenin bağlanmasını sağlayın.
3. Proje dosyalarını Visual Studio IDE'sinde açın.
4. Projeyi derleyin ve çalıştırın.

## Kullanım

- Yeni envanter öğeleri eklemek için "Yeni Öğe Ekle" butonunu kullanın.
- Envantörde arama yapmak için "Arama" kutusunu kullanın ve sonuçları görüntüleyin.
